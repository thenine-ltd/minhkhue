== Description ==

Plugin này dành riêng cho khách hàng sử dụng dịch vụ của eSMS, giúp quý khách gửi tin nhắn vào số điện thoại của khách hàng khi sử dụng Contact Form 7, NinjaForms hoặc Woocommerce.

= Include =

This plugin includes API of eSMS service and has the following functions:

- [API Get Balance](https://esms.vn/SMSApi/ApiGetBalance)
- [API Send SMS](https://esms.vn/SMSApi/ApiSendSMSBrandname)
- And more infor here [eSMS API DOCS](https://esms.vn/sms-api)

So this plugin use your ApiKey and your SecretKey on eSMS to get balance and send sms.

== Installation ==

Các bước cài đặt thủ công:

1. Trước tiên tải plugin này về và có thư mục tên `esms`. Hãy tải nó lên `/wp-content/plugins/`.
2. Sau đó vào `Quản lý plugins` và `kích hoạt` plugin `eSMS - Gửi tin nhắn sms`.
3. Cơ bản đã xong. Bạn có thể vào `Settings / Cài đặt eSMS` trong menu của admin để chọn thêm nhiều tùy chọn khác nhau.
